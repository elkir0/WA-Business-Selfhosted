## What this project does

This project provides an open-source, self-hostable gateway that communicates with the **publicly documented WhatsApp Business Cloud API** offered by Meta. It is a third-party client of that public API. It does not include, redistribute, or imitate any proprietary WhatsApp client software, WhatsApp Business App code, or WhatsApp Web code.

## Naming

The acronym "WA" in the project name stands for "WhatsApp Business Cloud API" in a purely descriptive sense — it identifies what the gateway talks to. This is comparable to how third-party Twitter clients have historically named themselves (and how libraries such as `node-fetch` or `axios` refer to the protocols they implement).

## Logo and visual identity

The project's logo (a square rounded chat bubble containing a chevron, paired with a wordmark) was designed independently. It does not replicate any official Meta or WhatsApp mark, palette specifics, or trade dress.

## Color palette

Our primary brand color is `#0EA5E9` (sky-500 in the Tailwind palette), chosen deliberately to be distinct from Meta's `#1877F2` and from WhatsApp's `#25D366`. The visual style is intended to evoke "modern selfhosted infrastructure tool" rather than to mimic Meta or WhatsApp.
